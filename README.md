## 🚀 Featured Projects

### 🏥 AI-Powered Hospital Appointment Booking Agent
> Conversational agentic AI that autonomously manages hospital appointments via tool-calling.
- Architected an agentic system giving the LLM controlled DB access through a custom tool-calling interface for autonomous **create / retrieve / update / delete** on appointment records.
- Engineered detailed system instructions for reliable, in-scope responses with **zero hallucinated actions**.
- Built multi-turn conversational flows that proactively collect patient details before any DB operation.
- Designed a **dual-layer safety architecture**: guardrails controlling what reaches the LLM + a defensive prompting layer blocking adversarial queries.
- **Stack:** `Flask` · `SQLAlchemy` · `MySQL` (Backend) | `React.js` (Frontend)

### 📚 RAG-Based Enterprise Knowledge Assistant
> Multimodal RAG pipeline for Q&A over PDFs and YouTube transcripts.
- Built with **LangChain** document loaders and recursive text splitters for context-preserving chunking.
- Generated semantic embeddings via HuggingFace **multilingual-e5-large**, stored in **ChromaDB**; implemented **Similarity Search** and **MMR** retrieval strategies.
- Integrated **Groq-hosted LLaMA 3.1** with custom prompting for low-latency, hallucination-reduced, context-grounded responses.
- Added audio transcription via **Faster-Whisper** to extend the pipeline to video/podcast content, plus multilingual query support.
